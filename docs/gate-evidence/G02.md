## Contained child-process disclosure oracle

G01 recorded that W02 privacy tests must add their own contained child-process output oracle before
generating additional sensitive-shaped values. The identity and log-wire subprocess integration tests
satisfy this: each spawns an isolated interpreter, bounds child stdout/stderr, asserts an empty
stderr and a byte-exact stdout, and fails closed (`SystemExit(70)`) rather than emitting a traceback,
so a planted value can only be observed through the asserted, bounded channel.
