# Claude Code Host Guide

Read and follow `AGENTS.md`; it is the canonical repository instruction. The five canonical project
skills live in `skills/`: `milhouse-ops`, `milhouse-feedback`, `milhouse-gate-review`,
`milhouse-compound`, and `milhouse-oss-maintainer`. Invoke them through the host's supported skill
mechanism or read the canonical `SKILL.md` directly when discovery aliases are unavailable.

Use the Milhouse MCP server only after its owning gate has passed and it is explicitly configured.
All privacy, delegation, mutation, verification, and release authority remains defined by `AGENTS.md`
and `docs/implementation-status.md`.
